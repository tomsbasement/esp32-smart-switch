# ESP32 Smart Switch with presence detector
Smart switch based on ESP32 and presence detector (SEN0609) for Esphome

Credits : Thanks to "Smart Home Scene" for their article and configuration which made this project easier to make.
Here is their original article : https://smarthomescene.com/diy/diy-presence-sensor-with-25m-detection-range-using-dfrobot-c4001-and-esp32/

## Required hardware

* ESP32-C6 from Dfrobot : https://www.dfrobot.com/product-2778.html?tracking=CH2ahHLno922IMb3XFammUzJDnc1pryXEVmuVTHKYGav95g0jDeqjLk90uqz3qqL
